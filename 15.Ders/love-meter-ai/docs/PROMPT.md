Create a React Native app using Expo and TypeScript.
Use expo skills.

App name: LoveMeter AI

Description:
A fun Valentine’s Day themed app that calculates relationship compatibility using AI.

Core features:

- Multi-step quiz (10 questions)
- Questions include multiple choice, slider, and short text input
- Collect user answers and send them to an AI model for analysis
- AI returns:
  - Compatibility percentage (0–100)
  - Short relationship analysis
  - One relationship advice
  - One motivational sentence

Screens:

1. Welcome Screen
   - App title
   - Short description
   - Start button

2. Quiz Screen
   - Progress indicator (e.g. Question 3/10)
   - Animated transitions
   - Different input types (radio, slider, text)

3. Loading Screen
   - Heart animation
   - “Analyzing your relationship…”

4. Result Screen
   - Compatibility percentage (big & animated)
   - AI analysis text
   - Advice section
   - Share button (share result as text)
   - Restart button

UI Style:

- Romantic
- Soft pink / red gradient
- Rounded cards
- Clean modern design
- Emoji support ❤️✨

Technical requirements:

- Expo + TypeScript
- Functional components
- useState and useEffect
- No backend required (mock AI response allowed)
- Well-structured components
- Clean folder structure
- Ready for future AI API integration

Bonus:

- Add simple animation for percentage
- Use SafeAreaView
- Responsive layout
